# Instruções para o Claude Code neste repositório

## Commits

- Use sempre meu nome e email configurados no `git config` do repositório
- **NÃO** adicione a linha `🤖 Generated with [Claude Code]`
- **NÃO** adicione `Co-Authored-By: Claude <noreply@anthropic.com>`
- **NÃO** mencione Claude Code, AI, ou qualquer ferramenta no corpo do commit
- Mensagens de commit devem seguir o padrão Conventional Commits:
  `tipo(escopo): descrição curta em português`
- Corpo do commit, quando necessário, deve explicar o "porquê", não o "o quê"
